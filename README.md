# .github

This repository holds the **public profile** and **shared community health files** for the [Tokens United](https://github.com/Tokens-United) organization. GitHub uses these as the defaults for every repository in the org that doesn't define its own.

| File | Purpose |
|---|---|
| [`profile/README.md`](./profile/README.md) | The landing page shown on [github.com/Tokens-United](https://github.com/Tokens-United) |
| [`SECURITY.md`](./SECURITY.md) | How to report a vulnerability responsibly |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Community standards (Contributor Covenant) |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | How to contribute across our repositories |
| [`SUPPORT.md`](./SUPPORT.md) | Where to get help |
| [`.github/ISSUE_TEMPLATE/`](./.github/ISSUE_TEMPLATE) | Bug report & feature request forms |
| [`.github/PULL_REQUEST_TEMPLATE.md`](./.github/PULL_REQUEST_TEMPLATE.md) | Default pull-request checklist |
| [`.github/FUNDING.yml`](./.github/FUNDING.yml) | Funding/links shown on repos |
